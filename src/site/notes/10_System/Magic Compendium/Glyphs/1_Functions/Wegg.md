# Wegg
**Translation:** Remove / Dispel
**School/Menu:** Abjuration (Level 1)

A function glyph used to safely peel away or dismantle an active spell matrix.

- **[[10_System/Magic Compendium/Glyphs/1_Functions/Wegg\|Wegg]] [[10_System/Magic Compendium/Glyphs/1_Functions/Tusc\|Tusc]]**: Remove Deception (Dispel an illusion safely)
- **[[10_System/Magic Compendium/Glyphs/1_Functions/Wegg\|Wegg]] [[10_System/Magic Compendium/Glyphs/1_Functions/Scht\|Scht]]**: Remove Defend (Dismantle an enemy's shield)
