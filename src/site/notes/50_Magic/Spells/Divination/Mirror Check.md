### Mirror Check
*[[40_People/Factions/Cyorian Academy/Classes/Divination Class\|Divination Class]] (Tier 1)*
**Mana Cost:** 1

The mage uses any reflective surface as a remote viewing screen for a location within line of sight.

**System:** Roll [[50_Magic/Spells/Divination/Slow Perception\|Slow Perception]] + [[10_System/Schools/Divination/Divination\|Divination]] vs difficulty 6. Requires 1 success.
**On success:** Uses any reflective surface as a remote viewing screen for a location within line of sight.
**Overcharge:** Can be spent on standard enhancements (Duration, Scale) or to apply the *Hindered* Status Effect (disorientation).
**On failure:** mana is lost.
