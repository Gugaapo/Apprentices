# Lebn
**Translation:** Living Target
**School/Menu:** Abjuration (Level 1)

A target modifier glyph that anchors a shield or ward directly to a biological organism.

## Example Formations
- **[[50_Magic/Glyphs/1_Functions/Scht\|Scht]] [[50_Magic/Glyphs/2_Objects/Lebn\|Lebn]]**: Defend Living Target (Cast a personal shield on an ally)
- **[[50_Magic/Glyphs/1_Functions/Scht\|Scht]] [[50_Magic/Glyphs/2_Objects/Lebn\|Lebn]] [[50_Magic/Glyphs/4_Modifiers/Lage\|Lage]]**: Defend Living Target Layer (A multi-layered personal ward)
