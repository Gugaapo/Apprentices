# Werk
**Translation:** Work / Action
**School/Menu:** Animation (Level 3)

A modifier glyph that allows a construct to perform complex, multi-step physical labor.

- **[[50_Magic/Glyphs/1_Functions/Lebe\|Lebe]] [[50_Magic/Glyphs/2_Objects/Ding\|Ding]] [[50_Magic/Glyphs/4_Modifiers/Werk\|Werk]]**: Animate Object Work (Command tools to build a wall automatically)
