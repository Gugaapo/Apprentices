# Abwa
**Translation:** Down / Crush
**School/Menu:** Dimensionalism (Level 3)

A modifier glyph that displaces a target forcefully toward the ground.

## Example Formations
- **[[50_Magic/Glyphs/1_Functions/Bewg\|Bewg]] [[50_Magic/Glyphs/4_Modifiers/Abwa\|Abwa]]**: Move Down (Slam a flying target into the earth)
