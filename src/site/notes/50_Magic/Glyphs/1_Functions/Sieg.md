# Sieg
**Translation:** Close / Seal
**School/Menu:** Abjuration (Level 4)

A function glyph used to forcefully shut spatial tears, lock pathways, or contain energy.

## Example Formations
- **[[50_Magic/Glyphs/1_Functions/Sieg\|Sieg]] [[50_Magic/Glyphs/3_Characteristics/Tohr\|Tohr]]**: Close Gateway (Forcefully shut an open portal)
- **[[50_Magic/Glyphs/1_Functions/Sieg\|Sieg]] [[50_Magic/Glyphs/3_Characteristics/Raum\|Raum]]**: Close Space (Lock down an area to prevent teleportation)
