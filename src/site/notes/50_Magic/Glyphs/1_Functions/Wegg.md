# Wegg
**Translation:** Remove / Dispel
**School/Menu:** Abjuration (Level 1)

A function glyph used to safely peel away or dismantle an active spell matrix.

## Example Formations
- **[[50_Magic/Glyphs/1_Functions/Wegg\|Wegg]] [[50_Magic/Glyphs/1_Functions/Tusc\|Tusc]]**: Remove Deception (Dispel an illusion safely)
- **[[50_Magic/Glyphs/1_Functions/Wegg\|Wegg]] [[50_Magic/Glyphs/1_Functions/Scht\|Scht]]**: Remove Defend (Dismantle an enemy's shield)
