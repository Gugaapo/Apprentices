# Bind
**Translation:** Graft / Fuse
**School/Menu:** Soul Carving (Soul Magic Specialization)

A function glyph used to permanently weld two disparate spiritual or physical entities together.

## Example Formations
- **[[50_Magic/Glyphs/1_Functions/Bind\|Bind]] [[50_Magic/Glyphs/2_Objects/Seel\|Seel]]**: Graft Soul (Combine two souls into one, or attach a soul to an object)
- **[[50_Magic/Glyphs/1_Functions/Bind\|Bind]] [[50_Magic/Glyphs/2_Objects/Mutr\|Mutr]]**: Graft Flesh (Create a chimera)
