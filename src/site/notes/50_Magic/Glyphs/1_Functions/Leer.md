# Leer
**Translation:** Void / Cancel
**School/Menu:** Negation (Abjuration Specialization)

A highly restricted function glyph that creates an absolute vacuum of mana, erasing magic from existence.

## Example Formations
- **[[50_Magic/Glyphs/1_Functions/Leer\|Leer]] [[50_Magic/Glyphs/2_Objects/Aura\|Aura]]**: Void Aura (Create a dead magic zone)
- **[[50_Magic/Glyphs/1_Functions/Leer\|Leer]] [[50_Magic/Glyphs/1_Functions/Schf\|Schf]]**: Void Create (Instantly delete a manifested projection spell)
