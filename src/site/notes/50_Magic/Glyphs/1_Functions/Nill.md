# Nill
**Translation:** Nullify / Counter
**School/Menu:** Abjuration (Level 2)

A violent function glyph used to aggressively shatter or counterspell an incoming magic attack.

## Example Formations
- **[[50_Magic/Glyphs/1_Functions/Nill\|Nill]] [[50_Magic/Glyphs/1_Functions/Schf\|Schf]]**: Nullify Create (Counter a projection spell mid-cast)
- **[[50_Magic/Glyphs/1_Functions/Nill\|Nill]] [[50_Magic/Glyphs/1_Functions/Bewg\|Bewg]]**: Nullify Move (Prevent a teleportation attempt)
