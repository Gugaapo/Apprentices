# Zwin
**Translation:** Compel
**School/Menu:** Mind Magic (Illusionism Specialization)

A restricted function glyph used to force a mental command or override a target's free will.

## Example Formations
- **[[50_Magic/Glyphs/1_Functions/Zwin\|Zwin]] [[50_Magic/Glyphs/2_Objects/Geis\|Geis]]**: Compel Mind (Force a target to obey a single command)
- **[[50_Magic/Glyphs/1_Functions/Zwin\|Zwin]] [[50_Magic/Glyphs/2_Objects/Geis\|Geis]] [[50_Magic/Glyphs/4_Modifiers/Gros\|Gros]]**: Compel Mind Greater (Total domination of a target's actions)
