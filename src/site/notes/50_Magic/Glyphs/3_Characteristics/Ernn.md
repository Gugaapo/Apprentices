# Ernn
**Translation:** Memory
**School/Menu:** Mind Magic (Illusionism Specialization)

A restricted characteristic glyph representing long-term memories, experiences, and identity.

## Example Formations
- **[[50_Magic/Glyphs/1_Functions/Sehn\|Sehn]] [[50_Magic/Glyphs/3_Characteristics/Ernn\|Ernn]]**: Perceive Memory (Extract a specific memory from a target)
- **[[50_Magic/Glyphs/1_Functions/Andr\|Andr]] [[50_Magic/Glyphs/3_Characteristics/Ernn\|Ernn]]**: Alter Memory (Erase or rewrite an event in the target's past)
