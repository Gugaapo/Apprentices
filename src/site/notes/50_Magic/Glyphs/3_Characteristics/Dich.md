# Dich
**Translation:** Density
**School/Menu:** Alteration (Level 2)

The characteristic glyph governing how tightly packed the molecules of an object are.

## Example Formations
- **[[50_Magic/Glyphs/1_Functions/Andr\|Andr]] [[50_Magic/Glyphs/3_Characteristics/Dich\|Dich]]**: Alter Density (Make an object heavier or lighter)
